Задания.

1. Реализовать сортировку строк файла.

Условия для оценки 3

Завести статический двумерный массив из STRINGS_COUNT строк и MAX_STRING_SIZE столбцов и динамический массив из STRINGS_COUNT строк. Массивы считаем параллельно проиндексированными, т.е. в элементах с одинаковыми индексами расположены соответствующие строки. Другие массивы для хранения строк использовать нельзя.
Считать из файла не более STRINGS_COUNT строк.
Если строка не больше MAX_STRING_SIZE расположить её в строке статического двумерного массива. Если больше - первые MAX_STRING_SIZE символов расположить в строке статического двумерного массива, а остальные в соответствующей строке динамического массива.
В итоге строки (в зависимости от длины) будут частично храниться в статическом массиве, частично в динамическом.
Реализовать сортировку строк, хранящихся описанным образом. Во время сортировки необходимо согласованным образом переставлять соответствующие части строк в двух массивах.
В функцию сортировки передавать указатель на функцию сравнения строк. Предложить и реализовать не менее четырёх разных функций сравнения строк (среди них обязательно функции сравнения в лексикографическом порядке с учетом и без учета регистра).
Для тестирования реализовать генерацию файла строк (не менее STRINGS_COUNT строк). Строки по длине должны быть меньше, равны и больше MAX_STRING_SIZE.

Шаблон для задания в каталоге template\1.

2. Реализовать арифметику больших чисел.

Условия для оценки 3

Большое число следует хранить в массиве 32-х битных машинных слов (unsigned int). В каждом слове (быть может, кроме последнего) должны храниться 32 бита большого числа. Бинарные алгебраические операции над большими числами должны реализовываться через последовательность действий над словами этих чисел.
Реализовать следующие операции:
- сложение и вычитание больших чисел;
- вывода большого числа;
- инициализации большого числа из строки шестнадцатиричных (десятичных по желанию) символов.
При реализации сложения и вычитания нельзя использовать тип long long (размером 8 байт).


Условия для оценки 4

Дополнительно реализовать операции умножения и деления больших чисел. При умножении и делении слов можно использовать тип long long.


Шаблон для задания в каталоге template\2.
